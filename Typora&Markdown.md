# Typora&Markdown

```
字体从大到小： 1~6个#		Ctrl+数字
有序列表： 1.
无序列表： *或者-或者+
加粗:  **需要加粗的字**				Ctrl+B
斜体字： *需要斜体的字*			Ctrl+I
斜体加粗字： ***需要斜体加粗的字***
删除线:用两个~开头，两个~结尾
下划线:<u>内容</u>   		Ctrl+U
链接:  [Title](URL) 		Ctrl+K
引用(引注):  >引用内容
建表： 键盘ctrl+t
图片： ![图片名](图片地址或者位置)
代码块:```
代码:用两个`在正常段落总表示代码
脚注:在需要添加脚注的文字后面+[+^+脚注+]
水平线:***
```
